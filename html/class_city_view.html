Append an option to a menu and bind it to the function CityView::OnAddTileMenuOption

All of the menu options to add a tile use the same menu handler, which uses a switch based on the ID to determine which option to make. This code cuts the number of lines of code in CityView::AddMenus by about half.

Parameters
mainFrameThe MainFrame object that owns the menu
