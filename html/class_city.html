std::shared_ptr< Tile > City::GetAdjacent ( std::shared_ptr< Tile >  tile,
int  dx,
int  dy 
)

Get any adjacent tile.

Given a tile in the city, this determines if there is another tile adjacent to it. The parameters dx, dy determine which direction to look.

The values for specific adjacencies (dx, dy, and direction):

  • -1 -1 Upper left
  • 1 -1 Upper right
  • -1 1 Lower left
  • 1 1 Lower right
Parameters
tileTile to test
dxLeft/right determination, -1=left, 1=right
dyUp/Down determination, -1=up, 1=down
Returns
Adjacent tile or nullptr if none.
